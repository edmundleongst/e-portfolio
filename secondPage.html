Introduction to Programming (First Modules)


The first module that I most eagerly wanted to share is the subject of Introduction of Programming. As I'd mentioned before, I'm interested in programming mostly. It is also part of my ambitious to be something of it. In YPC International College, they have teached me this subject and it is teached by Ms. Noraniza Jaafar. This module teaches me Java at that time and I'd know that it is a strong and standalone programming language. It provides the features of portability and exception handling. Portability is supported by a Java Virtual Machine (JVM). JVM helps in compiling the source code into a standard and platform independent bytecodes. JVM is a program that is highly optimized on each specific platform which you run your Java code. This means that each platform has one JVM that is installed specific to that operating system.

Another features of Java is that it has a class of Throwable. This class is used for exception conditions that the user program created should catch. This class is further divided into Exceptions and Error. The meaning of Exceptions is that it is a conditions which when running the application the application should try to catch while Error indicates a serious problem that the application cannot try to catch. One example of Exceptions is that when running the application if the result of one variable is not a number, application throws error and ask for data one more time. Another example of Error is that if a user have try to hack an application to steal the data inside. Application cannot try to catch this error from happening because it is unexpected and even if the application can a user can have more than one way to hack. In this situation, the primary concern when developing the application is to ensure that it can be run successfully rather than catch the user from hacking.

This in turned have helped to solved my problems and helped me in accomplishing the project.
